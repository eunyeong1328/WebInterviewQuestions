* get방식과 post방식의 차이는?
  Get방식 : 데이터의 값이 URL 뒤에 붙어 사용자에게 그대로 노출된다.
  Post방식 :  데이터가 body(본문) 에 포함되 전달되어 서버로 전송된다. URL이 노출되지 않아 보안에 강합니다.  
  Get은 가져오는 것(Select)이고 Post는 수행(Insert, Update, Delete)하는 것입니다.
